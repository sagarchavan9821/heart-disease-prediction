# heart-disease-prediction
# ❤️ Heart Disease Prediction App

This is a Machine Learning web app built using Streamlit that predicts the risk of heart disease based on user input.

## 🚀 Features
- User-friendly UI
- Real-time prediction
- KNN ML model
- Data preprocessing using scaler

## 🛠️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas

## ▶️ How to Run

1. Clone repo:
git clone https://github.com/sagarchavan9821/heart-disease-prediction.git

2. Install dependencies:
pip install -r requirements.txt

3. Run app:
streamlit run app.py

## 📊 Model
- KNN Classifier
- Scaled input data
